these are the four types of status:
    1.Modified: Changes detected in your file
    2.Staged: File is ready to be sibmitted
    3.Unmodified: No changes detected
    4.Untracked: New files that git doesnt yet track

To save the changes:
    1. git add (file name) #instead of file name put dot(.) all the files will be saved once
    2. git commit-m (some change)
    3. git push origin main
